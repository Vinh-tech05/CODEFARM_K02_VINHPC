# Các cách làm biến mất 1 phần tử trên giao diện.

## 1. Display: none

- Là thực sự biến mất khỏi giao diện.
- Kh thể tương tác.
- Kh chiếm chỗ.
- Kh nhìn thấy.
- Kh thể áp dụng đc animation

## 2. Visibility: hidden

- Là thực sự biến mất khỏi giao diện
- Kh thể tương tác.
- Có chiếm chỗ.
- Kh nhìn thấy.
- Kh thể áp dụng đc animation

## 3. opacity: 0

- Là làm cho phần tử tromg suốt.
- Có thể tương tác.
- Có chiếm chỗ.
- Kh nhìn thấy.
- Có thể áp dụng đc animation.
